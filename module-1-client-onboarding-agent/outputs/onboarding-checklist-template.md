# Output 4 — Onboarding Checklist (Template)

**Module 1 output format — Balance Operations Co. AI Bookkeeper Toolkit™**

**How to use:** The Agent fills this automatically. Use it as your project task list
to get the client fully live. You can ask the Agent to "add owners and due dates" or
paste it straight into your PM tool (Asana, ClickUp, Keeper, Financial Cents, etc.).

---

## Onboarding Checklist — {{CLIENT_NAME}}
**Firm:** {{FIRM_NAME}}  **Start date:** {{DATE}}  **Target go-live:** {{GO_LIVE_DATE}}

### Engagement setup
- [ ] Send engagement letter / proposal and get signed
- [ ] Collect first payment / set up billing
- [ ] Confirm scope, basis (cash/accrual), and cleanup period in writing
- [ ] Create client folder, naming conventions, and PM project

### Access & tools
- [ ] Receive accountant invite to {{ACCOUNTING_SOFTWARE}}
- [ ] Confirm bank & credit-card feed access (or statements)
- [ ] Get payroll provider access ({{PROVIDER}})
- [ ] Connect any POS / e-commerce / AP-AR tools
- [ ] Set up secure document portal for the client

### Books foundation
- [ ] Verify beginning balances tie to last filed tax return
- [ ] Review & clean chart of accounts
- [ ] Confirm subscription/plan supports needed features
- [ ] Set up classes/locations if needed (per industry profile)

### Cleanup kickoff
- [ ] Send Missing Documents List (Output 2) to client
- [ ] Begin Phase 1 of the Cleanup Plan (reconcile cash)
- [ ] Log open questions / "Ask My Accountant" items for client
- [ ] Schedule cleanup check-in cadence

### Communication
- [ ] Hold kickoff call; set expectations and response-time SLAs
- [ ] Introduce monthly reporting format and close timeline
- [ ] Confirm primary contact and preferred channel
- [ ] Add client to recurring task templates (transition to monthly service)
