# Output 1 — Cleanup Scope (Template)

**Module 4 output format — Balance Operations Co. AI Bookkeeper Toolkit™**

**How to use:** The Cleanup Project Agent fills this automatically. The scope is the
single best protection against scope creep — the "Out of scope" section is what you
point to when the client asks for "just one more thing." Attach it to (or fold it
into) the proposal and the engagement letter.

### Definition of done

- $0 in Uncategorized / Ask My Accountant; Opening Balance Equity cleared.
- Balance Sheet ties to source documents; P&L comparable month-over-month.
- {{Sales tax ties to filings}}; clean cutoff date established.
