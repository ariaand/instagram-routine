## Start here (15 minutes)

1. Read the top-level [`README.md`](../../README.md) for the big picture.
2. Open **`module-1-client-onboarding-agent/README.md`** and run the Client Onboarding
   Agent on your next prospect (or a test client).
3. Take that output into the **Cleanup Project Agent** (Module 4) to scope and price the work.
4. Use the **Proposal Generator** (Module 6) for the recurring monthly engagement.
5. Keep the **`prompt-library/`** open for quick one-off tasks.

> Each module has a copy-paste agent prompt, input templates, output templates, and a
> full worked example (the "Brightwave Café" client) so you can see exactly how it runs.
