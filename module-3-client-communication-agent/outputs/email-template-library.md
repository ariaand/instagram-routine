# Email Template Library

**Module 3 output — Balance Operations Co. AI Bookkeeper Toolkit™**

**How to use:** Find the email you need, fill the `{{PLACEHOLDERS}}`, read it once, and
send. Want it tailored to a specific situation or tone? Paste any template into the
Client Communication Agent and say "adapt this for {{situation}}."

> Keep them short. Owners skim. One clear ask per email.
